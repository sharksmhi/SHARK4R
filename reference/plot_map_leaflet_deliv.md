# Create a Leaflet map.

**\[deprecated\]** This function is deprecated and has been replaced by
[`plot_map_leaflet()`](https://sharksmhi.github.io/SHARK4R/reference/plot_map_leaflet.md).

## Usage

``` r
plot_map_leaflet_deliv(data, provider = "Esri.OceanBasemap")
```

## Arguments

- data:

  The data frame.

- provider:

  Tile provider, see
  https://leaflet-extras.github.io/leaflet-providers/preview/. Default
  is `Esri.OceanBasemap`.

## Value

HTML widget object.
